# 📊 Financial Data Analysis Dashboard (Power BI)

## 📝 Project Overview
This project involves a comprehensive analysis of a corporate financial dataset using **Power BI**. The goal is to transform raw financial data into actionable insights through interactive dashboards, helping stakeholders understand sales performance, profitability, and regional growth.

## 🚀 Key Features (The 5 Dashboards)
1. **Executive Summary:** High-level KPIs including Total Sales, Profit, and Units Sold.
2. **Geographical Analysis:** Global sales distribution across different countries (Canada, France, Germany, Mexico, USA).
3. **Product & Segment Performance:** Insights into which products and market segments (Government, Midmarket, etc.) drive the most revenue.
4. **Time Trend Analysis:** Monthly and yearly sales trends to identify seasonality.
5. **Discount & Efficiency:** Evaluating the impact of discount bands on net profit.

## 🛠️ Tools & Technologies
* **Power BI Desktop:** For data modeling and visualization.
* **Power Query:** For data cleaning and transformation (ETL).
* **DAX (Data Analysis Expressions):** For creating calculated measures and KPIs.
* **Dataset:** Financial Sample (Excel/CSV).

## 💡 Insights Discovered
* The **Government** segment represents the largest portion of sales.
* **France** and **Germany** show the highest profit margins in specific product lines.
